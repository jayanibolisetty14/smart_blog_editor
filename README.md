# smart_blog_editor
A Fullstack blog website built using MERN Stack 
This project implements modern authentication, form validation, user profiles, password reset functionality, and blog CRUD operations.
 Frontend
- React.js 
- Axios 
 Backend
- Node.js
- Express.js
- JWT Authentication
Database
- MongoDB
- Features
-  User Signup & Login
-  JWT Based Authentication
-  Forgot Password & Reset Password
-  Separate User Profiles
-  Create Blog Post
-  Edit Blog Post
-  Delete Blog Post
-  View All Blogs
